## Project Overview

The **Online Course Analysis** project focuses on analyzing recorded lecture data collected from multiple EdTech platforms. The purpose of this analysis is to help an EdTech startup better understand learner behavior, identify high-demand course categories, and optimize its recorded lecture offerings.

The Power BI dashboard delivers **category-wise insights** covering course distribution, learner engagement, skill relevance, language preferences, instructor performance, and accessibility features such as subtitles.

---

## Problem Statement

An EdTech startup aims to expand its recorded lecture services and improve learner engagement. Although the organization has collected large volumes of course data from various EdTech websites, it requires structured analysis to transform this raw data into meaningful insights.

As a **Data Analyst**, the responsibility is to clean, analyze, and visualize the data to support data-driven decision-making. The analysis helps the organization to:

* Improve course planning and launch strategies
* Enhance accessibility for diverse learner groups
* Align course content with learner preferences and current job market demand

The **primary focus of this analysis is category-wise performance and trends**.

---

## üéØ Project Objectives

* Analyze the distribution of courses across categories and sub-categories
* Identify viewer engagement trends using course view counts
* Determine the most in-demand skills across different categories
* Study language distribution and learner language preferences
* Evaluate the impact of subtitles on viewer engagement
* Identify top-performing instructors based on ratings
* Analyze the relationship between course duration and viewership
* Examine whether skill diversity influences course popularity
